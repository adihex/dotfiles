# Wallpaper Attribution

## Generated wallpapers (01-05)
These wallpapers were **generated algorithmically** using Python Pillow for this dotfiles setup.
They are **CC0 / public domain** — use freely, no attribution required.

### Previews
- `01-dark-gradient.png` — Dark bottom-to-top gradient with subtle wave noise
- `02-soft-glow.png` — Centered blue/purple soft glow on dark background
- `03-geometric-landscape.png` — Minimal sunset landscape with horizon and mountains
- `04-grain.png` — Solid dark color with fine film grain texture
- `05-wave.png` — Diagonal wave-patterned gradient

### Generation script
Run to regenerate:
```bash
python3 ~/dotfiles/scripts/generate-wallpapers.py
```
